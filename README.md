# Discord Canary Updater for Arch Linux
## Проблема

Пользователи Arch Linux, использующие официальную версию Discord Canary (tar.gz), сталкиваются с неприятной особенностью: при каждом запуске приложение может требовать скачать обновление. Вручную скачивать архив, распаковывать его и заменять файлы каждый день это утомительно и неудобно.
## Решение

Этот простой скрипт на Go решает проблему автоматизации. Он делает следующее:

- Скачивает актуальный архив discord-canary.tar.gz с официального сервера Discord.
- Распаковывает его.
- Заменяет старую версию на новую.

## Установка
1. Скачиваем последнюю версию https://github.com/Lazlo105/Discord-canary-updater-for-Arch-linux/releases/latest
2. Выдаем права на исполнение:
```
chmod +x discord-updater
```
3. Запускаем
